关于 json 文件

默认情况下，minapp init 生成的 json 文件的后缀都是 .cjson 文件，你的编辑器不一定支持。

这类文件表示支持注释的 json 文件，即你可以在文件中任意使用 /* */ 或 // 注释，minapp 中的所有 json 文件都支持注释。
